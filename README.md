# gorse
Data On Tap Inc. is a Full MVNO operating network 302 100 in Canada. This is DOT's third release named 'gorse' code repository. It includes advanced security and authentication, various connectivity and communication tools, network intelligence, beta network integration, reservations, connectivity discovery and setup flows, eSIM/iSIM support, end-user support tools, bluetooth mesh connectivity for fallback, bootstrap wireless connectivity with great control, extra  advice of charge and and quality, connectivity management tools. Rest OpenAPI 3.1 based. PostgresQL based DB. Integrations worh payments, messaging, mapping, catalogue, orchestration and many other useful extensions. MCP supported via our own efforts and woh great care at get-dot-eSIM.replit.app/mcp
