# gorse
Data On Tap Inc. is a Full MVNO operating network 302 100 in Canada. This is DOT's code repository. It includes advanced security and authentication, various connectivity tools, network intelligence, network reservations, connectivity setup flows with eSIM/iSIM, support tools, bluetooth mesh connectivity for fallback, bootstrap wireless connectivity, D2C satellite UX early designs, connectivity frameworks and concepts to build on. Rest OpenAPI 3.1 based. Postgres secure DB. MCP server.
