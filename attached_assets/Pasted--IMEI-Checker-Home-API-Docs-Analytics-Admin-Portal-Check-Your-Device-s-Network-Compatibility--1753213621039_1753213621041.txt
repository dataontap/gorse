Access our comprehensive API to integrate IMEI checking capabilities into your applications

Available Endpoints
POST
/api/v1/check
Check device compatibility by IMEI number

Example Response:


{
  "device": {
    "make": "Apple",
    "model": "iPhone 14 Pro",
    "imei": "123456789012345"
  },
  "capabilities": {
    "fourG": true,
    "fiveG": true,
    "volte": true,
    "wifiCalling": "limited"
  }
}
